# 腾讯云 AI 出海经营合规决策看板 v04

本版本在 v02 基础上重点优化展示质感：

- 修复 Streamlit 多选筛选标签仍显示红色的问题；
- 增加 `.streamlit/config.toml`，将主色调整为蓝色商业分析风格；
- 优化侧边栏、行动卡片、KPI 卡片、结论区和整体背景；
- 保留 v02 的经营诊断、客户优先级、标准化解决方案、行动清单和下载功能。

运行方式：

```bash
python -m streamlit run app.py
```

如果浏览器仍显示旧样式，请先停止 Streamlit（Ctrl+C），重新运行，并刷新浏览器页面。
